# Aocam Local Server

The local server will eventually provide:

- Camera registry.
- Health polling.
- Event aggregation.
- Multi-camera dashboard API.
- Optional backup coordination.

For Delivery 0, this component is a standard-library placeholder so it can run without installing FastAPI yet.

Run:

```bash
python3 src/aocam_server/main.py --host 127.0.0.1 --port 8765
```
